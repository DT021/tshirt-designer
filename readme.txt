	Awesome T-Shirt Designer for woocommenrce.

== Description ==
This plugin allows you to display an awesome T-Shirt Designer on your website via short-code anywhere in page,
add unlimited sticker(clip art) via custom post and display on designer and easy to use on t-shirt.
Also text can be use with fancy font family, font size and color.

* woocommerce ready.
* unlimited sticker or art.

* front and back image of tshirt.
* design preview.

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>T-Shirt Designer</strong>" activate it.<br />

After activate plugin you will see "T-shirt - Settings" menu at left side on WordPress dashboard.<br />

<br />
<strong>How to use on page.</strong><br />


Use following short-code to display designer anywhere in page content. `[tshirt_designer]`<br />

before starting you need to add some sticker via custom post "Sticker" with thumbnail image. 
and you will see option for uploading front and back image of tshirt of custom post "product"


Please watch this video tutorial

https://www.youtube.com/watch?v=w_2qdMQqNQQ
